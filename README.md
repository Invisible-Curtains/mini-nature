# Mini Nature - Garden Café & Nursery

A professional website for Mini Nature, a garden café and plant nursery in Kathmandu, Nepal.

## Features
- Responsive design (works on mobile and desktop)
- Menu showcase with pricing
- Nursery plant collection
- Contact form
- Green & brown nature theme

[View on GitHub Pages](https://invisible-curtains.github.io/mini-nature)

- GitHub Pages (free hosting)
